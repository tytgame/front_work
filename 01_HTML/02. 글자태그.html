그 밖의 글자를 다루는 태그들

일반 글꼴(태그로 감싸지 않은 글꼴)

b(bold): 글자를 두껍게 표시하는 태그

strong : 글자를 두껍게 표시하는 태그 + 스크린 리더에서 강조하여 읽어줌

i : 기울인 글꼴

em : 특히 강조할 때 기울임 글꼴로 표시 + 스크린 리더에서 강조하여 읽어줌

mark : 형광펜 효과를 주는 태그

u : 글자에 밑줄이 그어지는 태그

s : 글자에 취소선을 넣어주는 태그

small : 글자를 작게 해주는 태그

기본글자에 sub : 아래첨자를 넣어주는 태그
sup : 윗첨자를 넣어주는 태그

abbr : 줄임말 표시
